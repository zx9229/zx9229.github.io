## 操作  
```
$ hexo clean
  清除缓存文件 (db.json) 和已生成的静态文件 (public)。
$ hexo generate
  生成静态文件。
$ hexo deploy
  部署网站。
$ git add *
  将文件添加到索引库中。
$ git commit -m "message"
  提交修改内容。
$ git pull --rebase
  拉取最新的版本到本地。
$ git push
  将本地版本库的分支推送到远程服务器上。
```
